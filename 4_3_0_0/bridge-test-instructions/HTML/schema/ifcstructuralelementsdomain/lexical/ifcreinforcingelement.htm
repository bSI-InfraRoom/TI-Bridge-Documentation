Concept usage

Body AdvancedSwept DiskSolid PolyCurve Geometry

The Body AdvancedSwept DiskSolid PolyCurve Geometry concept applies to this entity.

The default geometric item to exchange the body shape representation of reinforcing elements is the IfcSweptDiskSolid using the IfcIndexedPolyCurve to describe the Directrix.



Element Voiding

The Element Voiding concept (defined at a supertype) does NOT apply to this entity.

The various subtypes of IfcReinforcingElement do not support the concept of voiding - there shall no voids (openings, cut-outs, recesses) being defined as shape features for reinforcing elements.
