4.8.1.9.1 Reference SweptSolid Geometry

The Reference SweptSolid Geometry is the reference representation of the 3D shape of a product by swept solid models, only allowing for the basic extruded area solids and revolved area solids. Being a reference representation it is normally not displayed and it is not used in a voiding relationship.

The following attribute values for the IfcShapeRepresentation holding this geometric representation shall be used:

  • IfcShapeRepresentation.RepresentationIdentifier = 'Reference'
  • IfcShapeRepresentation.RepresentationType = 'SweptSolid'
  • IfcShapeRepresentation.Items = IfcExtrudedAreaSolid, IfcRevolvedAreaSolid
